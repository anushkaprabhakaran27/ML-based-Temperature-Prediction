#Temperature Prediction System
## Project Overview
This project predicts the temperature using Machine Learning.it uses historical weather data and RandomForest Reggressor model to estimate temperature based on weather conditions.

## Feature
  -Predicts temperature using weather parameters
  -Uses Rondom Forest Regression
  -Evaluates model performance using
       --R**2 Score
       --Mean Absolute Error(MAE)
       --Mean Squared Error(MSE)
       --Root Mean Squared Error(RMSE)
   -Generates Actual vs Predicted Graph
   -Displays Feature Importance graph
   -Saves the trained model for future predictions

## Technologies used
   -Python
   -Pandas
   -Numpy
   -Scikit-learn
   -Matplotlib
   -Joblib

## Model Performance
   -R**2 Score: "0.9502"
   -MAE:"1.57 degree celsius"
   -RMSE:"1.96 degree celsius"
